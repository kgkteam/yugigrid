# YugiGrid 🧩

YugiGrid is a daily Yu-Gi-Oh! themed logic puzzle inspired by grid-based guessing games.

Each day a new 3×3 grid is generated where rows and columns define rules
(e.g. Level, Type, Attribute, Extra Deck mechanics).
Your goal is to place the correct cards so that **each card satisfies both its row and column rule**.

---

## 🎮 How to Play

- The grid is **3×3**
- Each row and column has a rule
- A card must match **both** rules of its cell
- A card can be used **only once**
- The puzzle is **daily**, generated from a deterministic seed

---

## 🛠️ Tech Stack

- Vanilla **HTML / CSS / JavaScript**
- Deterministic RNG (seeded daily)
- No backend required
- Hosted on **Netlify**
- Card data sourced from public Yu-Gi-Oh! databases
